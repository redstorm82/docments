browserify编辑

Browserify 可以让你使用类似于 node 的 require() 的方式来组织浏览器端的 Javascript 代码，通过预编译让前端 Javascript 可以直接使用 Node NPM 安装的一些库。[1] 
中文名
Browserify
外文名
Browserify

目录

▪ 安装
▪ 示例

安装

npm install -g browserify

示例

这是 main.js 的内容，像普通的 nodejs 程序那样使用 require() 加载库和文件：
var foo = require('./foo.js');
var bar = require('../lib/bar.js');
var gamma = require('gamma');
var elem = document.getElementById('result');
var x = foo(100) + bar('baz');elem.textContent = gamma(x);
导出的方法：
module.exports = function (n) { return n * 111 }
使用 browserify 编译：
$ browserify main.js > bundle.js
现在 main.js 需要的所有其它文件都会被编译进 bundle.js 中，包括很多层 require() 的情况也会一起被递归式的编译过来。
编译好的 js可以直接拿到浏览器使用
<script src="bundle.js"></script>
